# 📚 Book Haven

A responsive book store landing page built with HTML & CSS.

🔗 **Live:** https://shreya2005er-ctrl.github.io/bookstore-frontend/

## Features
- Responsive layout with Flexbox
- Hero section with CTA
- Book listing cards with image, title, price

## Tech Stack
HTML5, CSS3

## Run Locally
```bash
git clone https://github.com/shreya2005er-ctrl/bookstore-frontend.git
```
Open `index.html` in your browser.
